Scraper

Defined in lib/ScrapeService.js:45

This is the main method of this library. Scrape will accpt a string with the name of a roster, which will look to the "rosters" folder and load the JS file with the provided name. this can span multiple lines.

Parameters:

  • config Object | String

    This can be a string that contains the name of the roster to load or the actual roster config object.

  • [options] Objet optional

    Options for scrape

    • [autoload=false] Objet optional

      set to true if you want to call load immediately

    • [onLoad] Objet optional

      callback for load event.

Returns:

Scraper:

Returns true on success

Example:

var scrapeService = new ScrapeService(websiteLoader, consoleOutput);
var scrapeService.Scrape("NFL", {autoLoad: true}); //
